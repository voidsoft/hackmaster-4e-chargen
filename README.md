# Hackmaster 4th Edition Character Generator


# Viewing the Readme file in VSCode

To view the formatted version of this file click **Ctrl + Shift + V**

# Before you Start

You will need [Node.js installed](https://nodejs.org/), once this is installed you need to run:

```
npm install
```

This should install react and all the required modules into the project.

# Running the Code

To run the site on a local server use:

```
npm run dev
```

To run the unit tests, use:

```
npm run test
```
